1 如何C盘中的WSL2转移到D盘

将 WSL2 (Windows Subsystem for Linux 2) 从 C 盘迁移到 D 盘，最推荐的方法是使用 wsl --export 和 wsl --import 命令。

这个过程涉及四个核心步骤：导出、注销、导入和恢复默认用户。

第一步：查看所有已安装的 WSL 发行版及其状态，并确保它们已停止运行。

记下您想要迁移的发行版的名称（Ubuntu-22.04`）

如果状态不是 Stopped，运行此命令关闭所有 WSL 实例：

第二步：将 C 盘的发行版导出为 .tar 文件

将C盘内的wsl导出为一个 .tar 备份文件，暂时存放在 D 盘的某个位置。

  • 注意： 此步骤可能需要一些时间，具体取决于现在 WSL 系统大小。

第三步：注销原有发行版，原 C 盘上 WSL 占用的所有空间将被释放。

第四步：将 .tar 文件导入到 D 盘新位置

数据会存放在 VHDX 文件中，并保存在 D 盘目录。

注意： 必须添加 --version 2 以确保它是 WSL 2
